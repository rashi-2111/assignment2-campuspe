# Authentication Flow Website

This project is a simple multi-page authentication interface created using HTML and CSS.  
It demonstrates basic UI design, page structuring, and navigation flow typically seen in login systems.

## Pages Included

- Login  
- Register  
- Forgot Password  
- Reset Password  
- Dashboard  

## Key Features

- Clean pastel-themed design  
- Responsive centered layout  
- Styled input fields and buttons  
- Logical navigation between pages  

## Technologies Used

- HTML5  
- CSS3  

## Project Structure

project-folder/
│── login.html  
│── register.html  
│── forgot.html  
│── reset.html  
│── dashboard.html  
│── style.css  
│── README.md  

## How to Run

Open `login.html` directly in a web browser  

or  

Run using XAMPP  in the Browser:
http://localhost/folder-name/login.html
